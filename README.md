## Problem Statement

Synthetic Aperture Radar (SAR) imagery enables all-weather, day-and-night ground surveillance but presents unique classification challenges: multiplicative speckle noise, aspect-angle-dependent target signatures, and high inter-class visual similarity between vehicle types. This project implements a domain-engineered preprocessing pipeline combined with a lightweight CNN to perform Automatic Target Recognition (ATR) on the MSTAR benchmark dataset.

## Dataset

| Property | Value |
|---|---|

| Classes | BMP2, BTR70, T72 |

| Image size | 32×32 grayscale SAR chips |
| Preprocessed size | 64×64 (after pipeline) |
| Source | Downloaded automatically by `load_data.py` |

## Pipeline

```
Raw SAR Chip (32×32)
       │
       ▼
[1] Lee Adaptive Filter       — adaptive smoothing for multiplicative speckle noise
       │
       ▼
[2] CFAR Segmentation         — percentile-based threshold to isolate target return
       │
       ▼
[3] Morphological Cleanup     — binary opening/closing to remove stray noise pixels
       │
       ▼
[4] Standardised Centre Crop  — 64×64 crop centred on target region
       │
       ├──▶ [5a] HOG + RBF-SVM    (classical baseline)
       └──▶ [5b] SARNet CNN       (deep learning classifier)
```

## Results

| Model | Test Accuracy | BMP2 F1 | BTR70 F1 | T72 F1 |
|---|---|---|---|---|
| HOG + SVM | 91.4% | 0.86 | 0.89 | 0.99 |
| SARNet CNN (10 epochs) | 90.1% | 0.84 | 0.87 | 0.99 |

### CNN Training Curve

| Epoch | Train Acc | Test Acc |
|---|---|---|
| 1 | 46.6% | 33.3% |
| 4 | 86.2% | 59.6% |
| 6 | 94.8% | 91.2% |
| 9 | 96.8% | 95.6% (peak) |
| 10 | 97.8% | 90.1% |

### Key Observations

- T72 achieves F1 = 0.99 in both models due to a distinctive turret SAR signature
- BMP2 ↔ BTR70 confusion is the dominant error — both are wheeled armored vehicles with similar radar cross-sections at overlapping aspect angles
- Preprocessing pipeline is critical — Lee filter and CFAR segmentation are necessary for viable accuracy in both classifiers

## Model Architecture — SARNet

```
Input:        (N, 1, 64, 64)
Conv Block 1: Conv2d(1→32,   3×3, pad=1) → BatchNorm2d → ReLU → MaxPool2d(2×2)
Conv Block 2: Conv2d(32→64,  3×3, pad=1) → BatchNorm2d → ReLU → MaxPool2d(2×2)
Conv Block 3: Conv2d(64→128, 3×3, pad=1) → BatchNorm2d → ReLU → MaxPool2d(2×2)
Flatten:      128 × 8 × 8 = 8192
FC1:          Linear(8192→512) → ReLU → Dropout(0.5)
FC2:          Linear(512→128)  → ReLU → Dropout(0.5)
FC3:          Linear(128→3)    → softmax output

Optimizer:    Adam (lr=0.001, StepLR decay ×0.5 every 5 epochs)
Loss:         CrossEntropyLoss
Batch size:   32
Epochs:       10
```
